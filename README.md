# Simple Banking Application

This project implements a basic **Banking System** in Java using Object-Oriented Programming principles.  
It was developed as part of internship tasks to practice encapsulation, method design, and error handling.

---

## Features
- Deposit money into an account
- Withdraw money with error handling for insufficient funds
- Check account balance
- Console-based menu interface
- In-memory account simulation

---

## Project Structure
simple-banking-application/
├── BankAccount.java   # Encapsulates account details and methods
├── Main.java          # Console interface for user interaction

# Simple Banking Application

This project implements a basic **Banking System** in Java using Object-Oriented Programming principles.  
It was developed as part of internship tasks to practice encapsulation, method design, error handling, and multi-user session handling.

---

## Features
- User must log in with their **account number** before using banking services
- Deposit money into an account
- Withdraw money with error handling for insufficient funds
- Check account balance
- Logout to allow another user to log in
- Exit system completely
- Console-based menu interface
- In-memory account simulation using `HashMap`
